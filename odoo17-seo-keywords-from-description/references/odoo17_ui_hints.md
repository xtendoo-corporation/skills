# Pistas de UI en Odoo 17 (para encontrar los campos)

## Campo description_ecommerce
Suele estar en:
- Pestaña “eCommerce” / “Website”
- Bloque “Descripción del sitio web”
- Editor HTML/WYSIWYG

## SEO
Suele aparecer como:
- Botón “Optimizar SEO”
- Acción “SEO”
- “Promocionar”
- A veces dentro de opciones relacionadas con Website

Si no aparece:
- Comprobar que Website/eCommerce está instalado.
- Comprobar permisos del usuario.
- Comprobar que el producto está publicado.

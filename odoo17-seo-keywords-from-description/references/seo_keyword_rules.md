# Reglas de Keywords SEO (Odoo eCommerce)

- 8–15 keywords máximo.
- Separación: coma + espacio.
- Priorizar intención de compra y long-tail:
  - “camiseta cofrade sevilla”
  - “souvenir virgen macarena”
- Incluir términos locales si aplica (ciudad, hermandad, evento).
- Evitar keywords genéricas sin intención (“producto”, “oficial”, “tienda online”) salvo que aporten contexto real.
- Si hay entidad/marca/hermandad en el texto, incluirla.
- Mantener consistencia: minúsculas.
